# Taoli Tools 使用手册

这是 Taoli Tools 的官方使用手册文档站点，使用 VitePress 构建。

## 本地开发

```bash
# 安装依赖
npm install

# 启动本地开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview

# 部署到 GitHub Pages
npm run deploy
```
