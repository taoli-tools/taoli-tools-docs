# 安装部署

## 本地运行

本地电脑打开网页， [解除浏览器跨域限制](%E8%A7%A3%E9%99%A4%E6%B5%8F%E8%A7%88%E5%99%A8%E8%B7%A8%E5%9F%9F%E9%99%90%E5%88%B6.md)  后就可以直接使用。

## 服务器运行

也可以在服务器上运行，并使用远程桌面连接服务器。

> [!WARNING]
> 为了资金安全，请尽量选择大牌厂商的服务器，比如阿里云、腾讯云、AWS
