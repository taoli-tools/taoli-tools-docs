# 本地运行

本地电脑打开网页，[解除浏览器跨域限制](../disable-browser-cors/) 后就可以直接使用。

## 推荐浏览器

建议使用以下浏览器以获得最佳体验：

- **Google Chrome**（推荐）
- **Microsoft Edge**（基于 Chromium）

## 使用步骤

1. 访问 [https://taoli.tools](https://taoli.tools)
2. 按照 [解除浏览器跨域限制](../disable-browser-cors/) 的说明配置浏览器
3. 刷新页面即可正常使用

## 注意事项

- 本地运行无需安装任何额外软件
- 工具完全在浏览器中运行，所有数据保存在本地
- 不会收集任何密钥或交易相关的数据
