在 [https://taoli.tools/settings](https://taoli.tools/settings) 页面填写交易所的API信息，然后点击Save按钮保存

点击🔄按钮可以测试到交易所的延迟，最好不要超过150ms，如果超过，可以考虑 [Windows Server](%E5%AE%89%E8%A3%85%E9%83%A8%E7%BD%B2/Windows%20Server.md) 

> [!WARNING]
> 建议开启 API Key 的 IP 白名单功能，以提高安全等级。

> [!WARNING]
> 一定**不要**开启 API Key 的转账和提现功能。
