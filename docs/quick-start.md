# 快速开始

1. 访问 [交易页面](https://taoli.tools/trade)，点击蓝色按钮，使用 Telegram 登陆。
2. 点击 [交易页面](https://taoli.tools/trade) 页面右上角进入 [设置页面](https://taoli.tools/trade?settings=1)
3. 将左上角 UID 后面的数字发给 [Aliez Ren](https://t.me/aliez_ren)，付款后，给你开通 VIP（VIP0 无需这一步）。
4. 在 [设置页面](https://taoli.tools/trade?settings=1) 添加交易所的 API Key 和 Secret。然后点击 Save 按钮测试是否正确。
5. 在 [交易页面](https://taoli.tools/trade) 下方，「Symbol, CA」中搜索代币对名称，然后选择 A 和 B 代币对，然后点击「Add Pair」。
6. 后续步骤参考文档使用说明。建议仔细看文档，然后使用小额度操作尝试。
