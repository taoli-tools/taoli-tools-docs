# 安装部署

Taoli Tools 提供两种运行方式，根据你的需求选择合适的方案。

## 运行方式

### [本地运行](installation/local-setup.md)

**推荐新手使用**。在本地电脑浏览器中直接运行，无需额外部署。

- ✅ 简单快捷，无需额外配置
- ✅ 数据保存在本地，更安全
- ⚠️ 需要保持电脑开机和浏览器运行

### 服务器运行

适合需要 7×24 小时运行或降低网络延迟的用户。在云服务器上运行，通过远程桌面连接使用。

> [!WARNING]
> 为了资金安全，请尽量选择大牌厂商的服务器，比如阿里云、腾讯云、AWS

#### [Windows Server](installation/windows-server.md)

**推荐大多数用户使用**。使用 Windows 服务器，通过远程桌面连接。

- ✅ 操作简单，与本地 Windows 使用体验一致
- ✅ 可选择靠近交易所的服务器位置，降低延迟
- ⚠️ 需要付费购买服务器

#### [Linux Server](installation/linux-server.md)

适合有 Linux 使用经验的用户。使用 Docker 容器运行，通过 VNC 远程访问。

- ✅ 资源占用少
- ⚠️ 需要 Linux 使用经验
- ⚠️ 配置相对复杂
